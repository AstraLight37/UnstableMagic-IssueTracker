# 💥 Unstable Magic - Issue Tracker

Welcome to the official issue tracker for **Unstable Magic**, a highly destructive addon for the *Iron's Spells 'n Spellbooks* Minecraft mod.

This repository is strictly dedicated to tracking bugs, crashes, and community suggestions. **The mod's source code is not hosted here.**

## 🐛 How to Report a Bug or Crash
If you found a bug or your game crashed while using Unstable Magic, please open a new issue in the **[Issues](../../issues)** tab. 

To help me fix the problem as quickly as possible, please provide the following information:
*   **Minecraft Version** (e.g., 1.21.1)
*   **NeoForge Version** (e.g., 21.1.230)
*   **Unstable Magic Version**
*   **Iron's Spells 'n Spellbooks Version**
*   **Steps to Reproduce:** What exactly were you doing when the bug happened? (e.g., "I cast Orbital Strike Nuke near a village and the game froze").
*   **Crash Report / Latest Log:** Please DO NOT paste the entire crash log directly into the text. Upload your log to [Pastebin](https://pastebin.com/) or [mclo.gs](https://mclo.gs/) and share the link here.

## 💡 Feature Requests & Suggestions
Have a crazy idea for a new apocalyptic spell, a strange magical item, or an adjustment to existing spells? I'd love to hear it!

To suggest an idea:
1. Go to the **Issues** tab.
2. Create a new issue and add the `enhancement` or `suggestion` label (if possible).
3. Describe your idea in detail! What does the spell do? What is its rarity? How much mana should it cost?
